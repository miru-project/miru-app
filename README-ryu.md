## かんちく

### Flutterインストールすん

くまさんさ [Flutterこうしきドキュメント](https://flutter.dev/docs/get-started/install)

### いずんかんけいインストールすん

```bash
flutter pub get
```

### うぐかすん

```bash
flutter run
```

### パック

Android

```bash
flutter build apk
```

Windows

```bash
flutter build windows
```
